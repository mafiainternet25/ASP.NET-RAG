# 🎬 ASP.NET Cinema Booking System

## 📌 Overview

A simple cinema booking system built with ASP.NET Core.
Users can browse movies, book tickets, and manage their bookings.

---

## 🚀 Technologies

- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- JWT Login
- HTML, CSS, JavaScript
- Call API Key Chatbot

---

## 📂 Project Structure

- Controllers/ – handle API requests
- Models/ – data models
- Services/ – business logic
- Data/ – database context
- Views/ – frontend pages
- wwwroot/ – static files

---

## ⚙️ Setup & Run

```bash
git clone https://github.com/mafiainternet25/ASP.NET.git
cd ASP.NET
dotnet restore
dotnet run
```
